# スケルトンUSB充電器 回路シミュレーター

スケルトン（透明）筐体のUSB充電器アダプタ内部の回路を画像解析から推定し、  
各部品にかかる電圧・電流の時間変化をPythonでシミュレーション・可視化するプロジェクトです。

---

## 概要

| 項目 | 内容 |
|---|---|
| 入力 | AC 100V / 60Hz（日本規格・Type-A 2ピン） |
| 方式 | フライバックコンバータ（65kHz スイッチング） |
| 出力 | DC 5V / USB-A × 2ポート（Port1: 1.2A, Port2: 1.0A） |
| 回路推定根拠 | スケルトン筐体の実物写真（4方向）+ Gemini部品カウント |

---

## 推定回路構成

```
AC 100V/60Hz
    │
    ▼
[EMIフィルタ]  ← L1(コモンモードチョーク 1mH), L2(差動チョーク 470μH)
    │           ← C1(CXコンデンサ 0.1μF/275V), R1/R2(ブリーダー 1MΩ×2)
    ▼
[全波整流ブリッジ]  ← D1/D2(シリコンダイオード VF=0.7V×2)
    │
    ▼
[バルクコンデンサ平滑]  ← C2(330μF/200V)
    │  ~130V DC (リップルあり)
    ▼
[フライバックコンバータ]
    ├─ Q1: スイッチングMOSFET
    ├─ T1: トランス QL015Z XOH 2501（巻数比 約1:15）
    ├─ R3: NTCサーミスタ（突入電流保護, 10Ω@25℃）
    ├─ R6: ゲート抵抗 1Ω
    ├─ R8: 電流検出抵抗 22Ω
    ├─ R10: スナバ抵抗 100Ω + C3(スナバC 1nF) + D5(ツェナー 5.1V)
    ├─ U3: PWM制御IC（推定: AP3706 相当）
    │   ├─ R4/R5: 帰還分圧（47kΩ / 10kΩ）
    │   └─ R7: ソフトスタート抵抗 100kΩ
    └─ R9: バイアス抵抗 4.7kΩ
    │
    ▼
[二次側整流・平滑]
    ├─ D3: ショットキーダイオード（VF=0.45V, Port1用）
    ├─ D4: ショットキーダイオード（VF=0.45V, Port2用）
    ├─ C6: 出力電解コンデンサ（ECOT 680μF/10V, Port1）
    ├─ C7: 出力電解コンデンサ（ECOT 680μF/10V, Port2）
    ├─ C4: ブートストラップC（100pF）
    ├─ C5: 補助電源平滑C（47μF）
    ├─ C8: Yコンデンサ（青色, 4.7nF, ノイズ対策）
    └─ L3: 出力フィルタインダクタ（10μH）
    │
    ├─▶ USB-A Port1  5V / 1.2A ── R11(10Ω ダンピング)
    └─▶ USB-A Port2  5V / 1.0A ── R12(10Ω ダンピング)
```

## シミュレーション内容

### 全グラフ版（`usb_charger_all_graphs.py`）
- AC入力〜USB出力まで全段の電圧波形
- 抵抗 R1〜R12 の電圧・電流
- コンデンサ C1〜C8 の電圧・充放電電流
- インダクタ L1〜L3 の電圧・電流
- ダイオード D1〜D5 の電流・順方向電圧
- NTCサーミスタの抵抗値変化（温度依存）
- トランス T1 の一次/二次電流
- MOSFET Q1 のゲート/ドレイン電圧
- 制御IC U3 のPWMデューティ比推移
- 変換効率の推定
- 定常状態の電力フロー棒グラフ

### アニメーション版（`usb_charger_animation.py`）
- 7チャンネルの波形をスクロール表示（時間窓: 約10ms）
- 各ノードの現在値をリアルタイムバー表示
- 現在値の数値テキスト表示

## シミュレーションパラメータのカスタマイズ

`usb_charger_all_graphs.py` の冒頭部分で変更可能です。

```python
F_AC    = 60.0        # AC周波数 [Hz] → 50Hzに変更可能
V_AC    = 100.0       # AC実効値 [V]
F_SW    = 65_000.0    # スイッチング周波数 [Hz]
V_OUT   = 5.0         # 出力電圧 [V]
I_LOAD1 = 1.2         # USB-A Port1 負荷電流 [A]
I_LOAD2 = 1.0         # USB-A Port2 負荷電流 [A]
```

## 免責事項

このシミュレーションは、スケルトン筐体の外観写真とGeminiによる部品カウントから**推定した回路**に基づく近似計算です。実際の部品定数・回路トポロジーとは異なる場合があります。実機の回路解析・修理等には使用しないでください。
